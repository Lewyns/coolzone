## 🚀 เริ่มต้นใช้งาน

### ขั้นตอนที่ 1: เปิดเว็บไซต์

เปิด Browser แล้วไปที่:
```
http://localhost:8000
```

หรือ URL ที่แอดมินให้มา

### ขั้นตอนที่ 2: อนุญาตการเข้าถึงตำแหน่ง

เมื่อเว็บโหลดเสร็จ จะมีข้อความขอสิทธิ์:

```
"localhost" ต้องการใช้ตำแหน่งของคุณ
```

👉 **กด "อนุญาต" หรือ "Allow"**

💡 **ทำไมต้องอนุญาต?**
- เพื่อแสดงตำแหน่งคุณบนแผนที่
- คำนวณระยะทางไปจุดพักร้อน
- แจ้งเตือนเมื่อเข้าใกล้พื้นที่เสี่ยง

### ขั้นตอนที่ 3: รอโหลดข้อมูล

ระบบจะโหลด:
- ✅ แผนที่
- ✅ สภาพอากาศปัจจุบัน
- ✅ จุดพักร้อนใกล้คุณ
- ✅ พื้นที่เสี่ยง
- ✅ พยากรณ์อากาศ 6 ชั่วโมง

⏱️ **ใช้เวลา:** 2-3 วินาที

## 🗺️ หน้าจอหลัก

### 1. แผนที่ (ตรงกลาง)

```
┌─────────────────────────────────┐
│                                 │
│         [แผนที่]                │
│                                 │
│  📍 = คุณ                       │
│  🏢 = จุดพักร้อน                │
│  ⚠️ = พื้นที่เสี่ยง              │
│                                 │
└─────────────────────────────────┘
```

**การใช้งาน:**
- **ลาก** - เลื่อนแผนที่
- **Scroll** หรือ **Pinch** - ซูมเข้า/ออก
- **คลิกหมุด** - ดูรายละเอียด

### 2. แผงซ้าย (โปรไฟล์ + แนะนำ)

#### 👤 เลือกกลุ่มเสี่ยง
```
○ ทั่วไป (General)
● ผู้สูงอายุ (Elderly) ✓
○ คนทำงานกลางแจ้ง (Worker)
```

**เลือกตามกลุ่มของคุณ:**
- **ทั่วไป** - คนทั่วไป อายุ 18-60 ปี
- **ผู้สูงอายุ** - อายุ 60+ ปี ทนร้อนได้น้อยกว่า
- **คนทำงานกลางแจ้ง** - ทำงานกลางแจ้ง ต้องระวังเป็นพิเศษ

#### 🏢 จุดพักร้อนใกล้คุณ (Top 3)
```
1. สยามพารากอน 🏢
   📍 1.2 km (~15 นาที)
   ⏰ 10:00-22:00

2. BTS สยาม 🆓
   📍 0.8 km (~10 นาที)
   ⏰ 06:00-24:00

3. ห้องสมุดประชาชน 🆓
   📍 1.5 km (~19 นาที)
   ⏰ 08:00-20:00
```

👆 **คลิกการ์ด** → แผนที่จะซูมไปที่จุดนั้น

#### 🔍 ตัวกรอง

**กรองตามประเภท:**
```
[ทั้งหมด] [ฟรี] [เสียค่าใช้จ่าย]
```

- **ทั้งหมด** - แสดงทุกจุด
- **ฟรี** - เฉพาะจุดฟรี (BTS, MRT, ห้องสมุด, สวน)
- **เสียค่าใช้จ่าย** - เฉพาะห้างสรรพสินค้า

**กรองตามระยะทาง:**
```
[▼ < 2 km         ]
```

- **ทั้งหมด (< 5 km)** - แสดงทุกจุด
- **< 1 km** - เดินไปถึงได้ใน 12 นาที
- **< 2 km** - เดินไปถึงได้ใน 25 นาที
- **< 3 km** - เดินไปถึงได้ใน 37 นาที

### 3. แผงขวา (สภาพอากาศ + พยากรณ์)

#### 📍 GPS Status
```
✅ ตำแหน่งพบแล้ว
13.7563, 100.5018
```

#### 🌡️ ระดับความเสี่ยง
```
🔴 อันตราย

Heat Index: 45.6°C
อุณหภูมิ: 34.1°C
ความชื้น: 67%
───────────────────
อันตราย - หลีกเลี่ยง
กิจกรรมกลางแจ้ง

• ดื่มน้ำเป็นประจำ
• หลีกเลี่ยงแดดจัด
```

**ระดับความเสี่ยง:**
- 🟢 **ปลอดภัย** - สามารถทำกิจกรรมกลางแจ้งได้ตามปกติ
- 🟡 **ระวัง** - ระวังเมื่ออยู่กลางแจ้งนาน ดื่มน้ำบ่อยๆ
- 🔴 **อันตราย** - หลีกเลี่ยงกิจกรรมกลางแจ้ง หาที่ร่มเงา

#### 📊 พยากรณ์ 6 ชั่วโมง
```
19:00          🔴 HIGH
34.1°C · 67%   45.6°C

22:00          🟡 MEDIUM
31.3°C · 65%   38.1°C

01:00          🟢 LOW
26.9°C · 70%   28.7°C
```

🔄 **ปุ่มรีเฟรช** - อัปเดตข้อมูลใหม่

### 4. เมนูล่าง

```
[📍]  [🏢]  [⚠️]  [ℹ️]
ตำแหน่ง จุดพักร้อน พื้นที่เสี่ยง ข้อมูล
```

- **📍 ตำแหน่งฉัน** - กลับไปตำแหน่งของคุณ
- **🏢 จุดพักร้อน** - เปิด/ปิดหมุดจุดพักร้อน
- **⚠️ พื้นที่เสี่ยง** - เปิด/ปิดหมุดพื้นที่เสี่ยง
- **ℹ️ ข้อมูล** - แสดงข้อมูลแอป

## ✨ ฟีเจอร์ต่างๆ

### 1. ดูสภาพอากาศปัจจุบัน

**วิธีใช้:**
1. เปิดแอป → ระบบโหลดอัตโนมัติ
2. ดูแผงขวาบน

**ข้อมูลที่ได้:**
- อุณหภูมิจริง (°C)
- ความชื้น (%)
- Heat Index (อุณหภูมิที่รู้สึก)
- ระดับความเสี่ยง + คำแนะนำ

### 2. เปลี่ยนกลุ่มเสี่ยง

**วิธีใช้:**
1. ดูแผงซ้าย → "👤 เลือกกลุ่มเสี่ยง"
2. คลิกเลือกกลุ่มของคุณ

**ผลลัพธ์:**
- ระดับความเสี่ยงเปลี่ยนตามเกณฑ์กลุ่ม
- คำแนะนำเปลี่ยนตามกลุ่ม
- พยากรณ์อัปเดตอัตโนมัติ

**ตัวอย่าง:**
```
อุณหภูมิเดียวกัน 35°C

ทั่วไป:      🟡 ระวัง
ผู้สูงอายุ:   🔴 อันตราย  (เกณฑ์ต่ำกว่า)
คนงาน:       🟡 ระวัง
```

### 3. หาจุดพักร้อนใกล้ที่สุด

**วิธีที่ 1: ดูแนะนำ Top 3**
1. ดูแผงซ้าย → "🏢 จุดพักร้อนใกล้คุณ"
2. เลือกจุดที่ชอบ

**วิธีที่ 2: ดูบนแผนที่**
1. ดูหมุดสีเขียว 🆓 (ฟรี) และสีน้ำเงิน 🏢 (เสียค่าใช้จ่าย)
2. คลิกหมุดดูรายละเอียด

**ข้อมูลที่ได้:**
- ชื่อสถานที่
- ประเภท (ฟรี/เสียค่าใช้จ่าย)
- ระยะทาง (km + นาทีเดิน)
- เวลาเปิด-ปิด

### 4. ใช้ตัวกรอง

**กรองตามประเภท:**
1. แผงซ้าย → "🔍 ตัวกรอง"
2. คลิกปุ่ม "ฟรี" หรือ "เสียค่าใช้จ่าย"

**กรองตามระยะทาง:**
1. เลือก Dropdown "ระยะทาง"
2. เลือกระยะที่ต้องการ

**ผลลัพธ์:**
- หมุดบนแผนที่กรองตามเงื่อนไข
- แนะนำ Top 3 เปลี่ยนตาม

### 5. ดูพื้นที่เสี่ยง

**วิธีใช้:**
1. คลิกปุ่ม "⚠️ พื้นที่เสี่ยง" ล่างซ้าย
2. หมุดสีแดง/ส้ม/เหลือง จะปรากฏ

**สีหมุด:**
- 🔴 **สีแดง** - อันตรายสูง
- 🟡 **สีส้ม** - ระวัง
- 🟢 **สีเหลือง** - ต่ำ

**คลิกหมุดดู:**
- ชื่อพื้นที่
- ระดับความเสี่ยง
- คำเตือน
- คำอธิบาย

### 6. รับการแจ้งเตือน

**เมื่อไหร่แจ้งเตือน:**
- เมื่อคุณอยู่ใกล้พื้นที่เสี่ยงสูง (< 500 เมตร)

**รูปแบบการแจ้งเตือน:**

**1. Browser Notification** (ถ้าอนุญาต)
```
⚠️ พื้นที่เสี่ยงใกล้คุณ!
ลานจอดรถคอนกรีต สยาม - หลีกเลี่ยงการยืนนาน...
```

**2. In-App Alert** (แสดงเสมอ)
```
┌────────────────────────────┐
│ ⚠️ พื้นที่เสี่ยงใกล้คุณ! ✕ │
│                            │
│ ลานจอดรถคอนกรีต สยาม      │
│ 450 m จากคุณ               │
│                            │
│ หลีกเลี่ยงการยืนนาน...     │
└────────────────────────────┘
```

**การจัดการแจ้งเตือน:**
- คลิก **✕** เพื่อปิด
- หรือรอ 10 วินาที จะหายเอง
- ไม่แจ้งเตือนซ้ำจุดเดิม

### 7. ดูพยากรณ์อากาศ

**วิธีใช้:**
1. ดูแผงขวาบน → "📊 พยากรณ์ 6 ชั่วโมง"
2. ดูข้อมูลแต่ละช่วง

**ข้อมูลแต่ละช่วง:**
- เวลา
- อุณหภูมิ + ความชื้น
- Heat Index
- ระดับความเสี่ยง

**สีพื้นหลัง:**
- สีแดง = อันตราย
- สีเหลือง = ระวัง
- สีเขียว = ปลอดภัย

**รีเฟรช:**
- คลิกปุ่ม "🔄 รีเฟรช" ขวาบน

## ❓ คำถามที่พบบ่อย (FAQ)

### Q1: GPS ไม่ทำงาน

**A:** ตรวจสอบ:
1. ได้อนุญาตสิทธิ์ GPS แล้วหรือยัง?
2. เปิด Location Services ในเครื่องหรือยัง?
3. ใช้ `http://localhost` หรือ `https://` หรือไม่? (GPS ไม่ทำงานกับ `file://`)

**วิธีแก้:**
- เปิดใหม่แล้วกด "Allow" เมื่อถามสิทธิ์
- ตรวจสอบ Settings → Privacy → Location

### Q2: ข้อมูลไม่อัปเดต

**A:** ลองทำตามนี้:
1. Refresh หน้าเว็บ (F5 หรือ Ctrl+R)
2. ตรวจสอบ Internet connection
3. คลิกปุ่ม "🔄 รีเฟรช" ใน Forecast Panel

### Q3: ไม่เห็นจุดพักร้อน

**A:** ตรวจสอบ:
1. คลิกปุ่ม "🏢 จุดพักร้อน" ล่างซ้าย (ต้องเปิด)
2. ตัวกรองอาจกรองออกหมด → เปลี่ยนเป็น "ทั้งหมด"
3. ระยะทางอาจไกลเกิน → เลือก "ทั้งหมด (< 5 km)"

### Q4: แจ้งเตือนไม่ขึ้น

**A:**
- **Browser Notification:** ต้องอนุญาตสิทธิ์ก่อน
- **In-App Alert:** แจ้งเฉพาะพื้นที่เสี่ยงสูงใกล้ๆ (< 500m)

### Q5: อุณหภูมิไม่ตรงกับแอปอื่น

**A:** เป็นเรื่องปกติเพราะ:
- CoolZone แสดง **Heat Index** (อุณหภูมิที่รู้สึก)
- แอปอื่นแสดงอุณหภูมิจริง
- Heat Index รวมความชื้นด้วย → มักสูงกว่าอุณหภูมิจริง

**ตัวอย่าง:**
```
อุณหภูมิจริง: 35°C
Heat Index: 45°C  (ร้อนขึ้นเพราะความชื้นสูง)
```

### Q6: ทำไมผู้สูงอายุได้ "อันตราย" แต่คนทั่วไป "ระวัง"?

**A:** เพราะผู้สูงอายุ:
- ทนความร้อนได้น้อยกว่า
- มีความเสี่ยง Heat Stroke สูงกว่า
- **เกณฑ์ต่ำกว่า** เพื่อความปลอดภัย

### Q7: ใช้ฟรีหรือเปล่า?

**A:** **ฟรี 100%!**
- ไม่มีค่าใช้จ่าย
- ไม่ต้องสมัครสมาชิก
- ไม่มีโฆษณา

### Q8: ใช้ออฟไลน์ได้หรือไม่?

**A:** **ไม่ได้** ต้องมี Internet เพราะ:
- ดึงข้อมูลสภาพอากาศแบบเรียลไทม์
- โหลดแผนที่จาก OpenStreetMap

### Q9: ใช้บนมือถือได้หรือไม่?

**A:** **ได้!** 
- รองรับทั้ง iOS และ Android
- Responsive design ปรับตามหน้าจอ
- แนะนำเปิดด้วย Chrome หรือ Safari

### Q10: ข้อมูลอัปเดตบ่อยแค่ไหน?

**A:**
- **สภาพอากาศ:** อัปเดตเมื่อเปิดแอป / เปลี่ยน Profile
- **พยากรณ์:** อัปเดตเมื่อเปิดแอป / คลิก Refresh
- **GPS:** อัปเดตเมื่อเคลื่อนที่
